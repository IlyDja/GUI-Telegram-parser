Перед использованием нужно получить API_ID и API_HASH для доступа к чатам Telegram: https://my.telegram.org/auth?to=apps
Введите полученные API_ID и API_HASH в файл "api_id and api_hash.txt" (в папке с приложением) вместо звёздочек.

Перед использованием программы установите синхронизацию времени. На Windows это делается так: параметры даты и времени > устанавливать время автоматически > вкл

Если поиск будет происходить по "супергруппе", должна быть ссылка именно на эту группу, а не на её отдельную комнату.

Если ошиблись при вводе номера телефона/кода подтверждения, или если хотите войти под другим номером - удалите файл my_session.session
